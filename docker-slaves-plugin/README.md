Run docker-slaves-plugin from source
====================================

# Prerequisite

- Dependency
  - install docker
  - install JDK 1.8
  - install maven 3
- Optional
  - shadowsocks (socks5 proxy)
  - privoxy (convert socks5 to http proxy)
